## E2E 流程总览

```
[注册/登录] → [求职画像] → [能力测评] → [AI职业探索] → [岗位匹配]
                                                              ↓
[学习进度] ← [学习路径] ← [差距分析] ←─────────────────────────┘
     ↓
[简历优化] → [模拟面试] → [完成求职闭环]

[企业端]: [项目描述输入] → [AI解析岗位] → [候选人推荐]

[管理端]: [用户管理] → [技能词典维护] → [运营看板]
```
